  • Find the “control” columns in our counts object
  • Extract just the “control” column values for all genes
  • Calculate the average value per gene in these “control” column
control.inds <- metadata$dex == "control"
control.counts <- counts[ , control.inds ]
control.mean <- rowMeans(control.counts)

Now do the same for the “treated” columns. Then make a plot of control.mean vs treated.mean

treated.mean <- rowMeans( counts[, metadata$dex == "treated"])

For book-keeping let’s store these together as a new object called meancounts

We must often use log2 transform for this kind of data in bioinformatics because it makes the interpretation much easier.

# Treated / Control

log2(20 / 20 )
[1] 0
log2( 40/ 20)
[1] 1
log2( 10 / 20 )
[1] -1
log2( 80 / 20)
[1] 2

We call this little fraction the “log2 fold change” as it tells us how much or less gene expression we have in units of doubling, etc.

Let’s calculate the log2 fold change for our treatment.mean and control.mean counts and call this log2fc

meancounts$log2fc <- log2( meancounts$treated.mean / 
                           meancounts$control.mean )
