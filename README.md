Nano is just a basic text editor

Core unix commands include:

- pwd: Print working dir
- ls: List files in the current directory
- cd: Change directory
- mkdir: Make a new directory (folder)
- touch: Create a new file
- rm: Remove/Delete a file
- nano: A basic text editor
- mv:   Move/Rename a file
- cp:   Copy/Duplicate a file

**Key point:** pwd returns the "absolute path", when you are 
relative to the root of the file system. eg:
`/Users/raneemkassar/Desktop/class16`

Useful shortcuts include

- ../ Parent dir
- ~/  Home area
- * Wild card match everything
